## Typography System

### Primary Font: **Inter** (Sans-serif)
- **Psychology:** Modern, clean, professional, highly readable
- **Demographics:** Appeals to tech-savvy 25-55 age group
- **Use Case:** Headings, UI elements, body text

- **Accessibility:** Designed for clarity at all sizes

### Monospace Font: **JetBrains Mono**
- **Psychology:** Technical, precise, developer-friendly
- **Demographics:** Appeals to developers and technical users
- **Use Case:** Code snippets, technical data, API responses
- **Why:** Excellent for displaying WCAG criteria, CSS selectors
