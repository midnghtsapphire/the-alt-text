## 🎨 What We Built

The **OZ Graphics API** transforms The Alt Text from a standard SaaS into a **white-label platform** where every client gets their own custom-branded experience automatically generated by AI.

### Key Innovation

Instead of using generic templates, **every client gets**:
- ✅ Custom 3D logo with glassmorphism effects
- ✅ Personalized landing page with their brand colors
- ✅ Unique feature icons matching their aesthetic
- ✅ Branded dashboard and reports
- ✅ Marketing materials (social media graphics, email headers)

**All generated automatically in under 60 seconds.**

## 🚀 How It Works

### 1. Client Onboarding Flow

```
Client Signs Up
    ↓
Provides: Company Name, Industry, Preferred Colors
    ↓
OZ Graphics API Generates Complete Brand Asset Set
    ↓
Assets Stored in S3 (client-specific paths)
    ↓
Landing Page & Dashboard Load with Client Branding
```

### 2. API Endpoints

#### Generate Logo Set
```typescript
POST /api/oz/graphics/generate-logo
{
  brandName: "Acme Corp",
  industry: "healthcare",
  style: "glassmorphism",
  primaryColor: "#2563EB",
  iconSymbol: "shield"
}

Response:
{
  logoMain: "https://s3.../acme-corp/logo-main.png",
  logoIcon: "https://s3.../acme-corp/logo-icon.png",
  logoHorizontal: "https://s3.../acme-corp/logo-horizontal.png",
  favicon16: "https://s3.../acme-corp/favicon-16.png",
  favicon32: "https://s3.../acme-corp/favicon-32.png"
}
```

#### Generate Landing Page Assets
```typescript
POST /api/oz/graphics/generate-landing-assets
{
  brandName: "Acme Corp",
  style: "glassmorphism",
  colorPalette: {
    primary: "#2563EB",
    accent: "#06B6D4",
    success: "#10B981"
  },
  features: [
    { name: "Automated Scanning", description: "AI-powered WCAG compliance checks" },
    { name: "Real-time Monitoring", description: "24/7 accessibility monitoring" },
    { name: "Instant Fixes", description: "One-click accessibility repairs" }
  ]
}

Response:
{
  heroBackground: "https://s3.../acme-corp/hero-bg.png",
  featureIcons: [
    { name: "Automated Scanning", iconUrl: "https://s3.../acme-corp/icon-scan.png" },
    { name: "Real-time Monitoring", iconUrl: "https://s3.../acme-corp/icon-monitor.png" },
    { name: "Instant Fixes", iconUrl: "https://s3.../acme-corp/icon-fix.png" }
  ],
  productIllustration: "https://s3.../acme-corp/product-illustration.png"
}
```

### 3. Dynamic Asset Loading

```typescript
// Landing page component
function Hero({ clientId }: { clientId: string }) {
  const { data: branding } = trpc.branding.getClientBranding.useQuery({ clientId });
  
  return (
    <section 
      style={{
        backgroundImage: `url(${branding.heroBackground})`,
        '--primary-color': branding.primaryColor,
        '--accent-color': branding.accentColor
      }}
    >
      <img src={branding.logoMain} alt={branding.brandName} />
      <h1 style={{ color: 'var(--primary-color)' }}>
        {branding.brandName} Accessibility Compliance
      </h1>
    </section>
  );
}
```

## 📊 Database Schema

```typescript
export const clientBranding = pgTable('client_branding', {
  id: serial('id').primaryKey(),
  userId: integer('user_id').references(() => users.id).notNull(),
  brandName: text('brand_name').notNull(),
  industry: text('industry'),
  style: text('style').notNull(), // glassmorphism, cyberpunk, minimalist, corporate, playful
  
  // Color palette
  primaryColor: text('primary_color').notNull(),
  secondaryColor: text('secondary_color'),
  accentColor: text('accent_color'),
  successColor: text('success_color'),
  warningColor: text('warning_color'),
  dangerColor: text('danger_color'),
  
  // Asset URLs
  logoMain: text('logo_main'),
  logoIcon: text('logo_icon'),
  logoHorizontal: text('logo_horizontal'),
  favicon16: text('favicon_16'),
  favicon32: text('favicon_32'),
  heroBackground: text('hero_background'),
  productIllustration: text('product_illustration'),
  dashboardMockup: text('dashboard_mockup'),
  
  // Feature icons (JSON array)
  featureIcons: jsonb('feature_icons'), // [{ name: string, iconUrl: string }]
  
  // Marketing materials (JSON array)
  marketingMaterials: jsonb('marketing_materials'), // [{ type: string, url: string, dimensions: { width, height } }]
  
  // Metadata
  generatedAt: timestamp('generated_at').defaultNow(),
  regenerationCount: integer('regeneration_count').default(0),
  lastRegeneratedAt: timestamp('last_regenerated_at'),
});
```
